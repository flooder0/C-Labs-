Лаба 0. Вендинговый автомат на C#

### Требования
- показать список товаров с ценой и остатком;
- принять монеты номиналом 1/2/5/10 ₽;
- купить выбранный товар, выдать сдачу или отказать, если денег не хватает;
- отменить покупку и вернуть внесённые монеты;
- режим администратора с PIN (по умолчанию 1234): пополнение товара, добавление новой позиции, пополнение монет, просмотр банка и сбор наличных.

### Что внутри
- все данные держатся в памяти: после перезапуска остатки и банк сбрасываются к значениям по умолчанию;
- сдача считается обычным жадным алгоритмом для монет 1/2/5/10 ₽;
- код находится в одном проекте `src/Vending.App` без дополнительных библиотек.

### Как запустить
Нужен установленный .NET 8 SDK.

```
dotnet run --project src/Vending.App/Vending.App.csproj
```

### Замечания
- тесты для этой версии не писались;
- можно менять список товаров прямо в коде `Program.cs`, если нужно другое начальное наполнение.
